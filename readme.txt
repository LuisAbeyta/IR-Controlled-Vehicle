##Project description##

The is consist of one file coded in c++ and uses an arduino microcontroller to control a vehicle. The main functionality of the vehicle is controlled through a IR receiver and remote. Other hardware included two motors, three photocells, and an ultra sonic sensor. The photocells were include to add a mode the navigates towards a light source. A ultrasonic sensor was include to interrupt the program if the vehicle move to close to a wall.
